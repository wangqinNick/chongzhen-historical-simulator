# SillyTavern 导入清单

本清单用于确认“最终一键复制包”之外的可导入文件是否齐备。

## 推荐导入顺序

1. 新建角色：优先导入 `sillytavern/崇祯模拟器_Narrator角色卡.json`；也可使用 `sillytavern/崇祯模拟器_Narrator角色卡草案.md` 或最终一键复制包中的角色卡段落。
2. Author's Note：复制 `sillytavern/当前状态_AuthorNote模板.md`；如果换年份，用 `sillytavern/开局剧本_AuthorNote合集.md` 中对应年份。
3. World Info / Lorebook：优先导入 `sillytavern/import/崇祯历史模拟器_完整Lorebook.json`。
4. 轻量替代：只导入 `核心Lorebook`；需要模块但不需要人物时导入 `全模块Lorebook`；只想补人物时导入 `人物Lorebook`。
5. 快速局势：`快速卡Lorebook` 包含地区、军队、派系高频卡，可单独导入，也已合入完整 Lorebook。
6. 开局剧本：`开局剧本Lorebook` 包含崇祯元年、二年、四年、七年、十年、十四年 Author's Note 开局块，也已合入完整 Lorebook。
7. 多角色群聊：把 `sillytavern/人物卡/json/` 下的人物 JSON 逐个导入。
8. Data Bank / RAG：放入最终一键复制包、快速卡、基准数据登记表和资料目录。

## 每次修改后运行

```powershell
powershell -ExecutionPolicy Bypass -File scripts\build_all.ps1
```

手工分步执行也可以，但推荐使用上面的一键构建脚本，避免漏掉开局剧本或 Narrator JSON。
