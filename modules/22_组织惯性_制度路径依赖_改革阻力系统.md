# 组织惯性、制度路径依赖、改革阻力系统

## 用途

定义旧制度受益者、学习成本、官僚惯性和改革落地难度。

## 核心规则

- 改革不是写出新制度就完成。
- 改革必须经过试点、培训、监督、纠错、奖惩、复制和常态化。
- 旧制度会吸收、磨损或扭曲新制度。
- 改革阻力不只来自反对者，也来自学习成本和执行惯性。
